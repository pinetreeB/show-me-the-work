조사 팩 마커를 정리해 다시 보고합니다.

**재현:** `python demo.py` → `2 + 3 = -1` (덧셈이어야 하는데 음수).

**가설 1:** `add`가 `a + b`가 아니라 `a - b`를 반환한다 (연산자 오류).
**가설 2:** import 경로/오타 문제다.
**가설 3:** 인자 순서가 뒤바뀌었다.

**증거:**
- 가설 1 → `calc.py:2`가 `return a - b`. `2 - 3 = -1`로 관측값과 정확히 일치. **채택.**
- 가설 2 → `from calc import add`는 정상 실행되어 값이 반환됨. 오류는 반환값에만 존재.
- 가설 3 → 인자는 `(2, 3)`로 정상 전달됨. 순서 문제라면 `3 - 2 = 1`이어야 하나 결과는 `-1`.

**기각:** 가설 2·3 — 재현 결과(`-1`)가 오직 `a - b` 연산자 오류로만 설명되므로 배제.

**수정:** `calc.py:2` `return a - b` → `return a + b`

**전후 검증:** 수정 전 `2 + 3 = -1` → 수정 후 `2 + 3 = 5`. 정상.
